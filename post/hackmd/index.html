HackMD

Overview

HackMD is a collaborative online editor for Markdown files. Markdown is a simplified markup language that allow you to quickly format your documents while you write. HackMD provides an editor tailored for Markdown that provides a live rendering of the formatting as you type. The site also allows for multiple users to work on the document simultaneously.

Website Link: hackmd.io Platform: Online (There is also a standalone application supported on Windows, MacOS and Linux)

Cost

HackMD is free to use for personal and public team use. For personal use you are limited to 3 private notes for collaboration. Public team use is limited to public notes only. If you need private notes, a Private Team membership is require at a cost of $10USD/month for the first four users and $5USD/month for each additional user.

However, HackMD also includes an open source version. If the cost of the hackmd.io website is too much, users can deploy a private instance of the tool on their own servers. Of course, this requires much more technical knowledge and support.

Advantages and Challenges

Advantages

  • Markdown basics can be learned in minute.
  • Allows for collaborative creating and editing of content.
  • Allows for users to highlight and add comments to the content.
  • Notes can also be converted to books or slides.

Challenges

  • Cost for private repositories (or servers if you run your own instance).

Learning Strategy

From (Walsh 2018) HackMD would be used in peer instruction. HackMD can be used collaboratively by students to generate class note, slides for presentation, or even articles and reports. In all cases the students would leveraging each others strengths to better understand the material.

Pedagogical function

HackMD would fall into the pedagogical categories of organizational technologies and communications and collaboration technologies. HackMD can be used a tool to help students organize note, research and other material. It also allows the user to convert them into books or slides if the material is better suited for these types of presentation. And because of its online access and ability to create teams, HackMD can easily be used as a tool for communications and collaboration.

Accessibility Concerns

No accessibility concerns identified.

Special Equipment Required

None unless you choose the self-hosted option. In that case a server and related network infrastructure is required.
